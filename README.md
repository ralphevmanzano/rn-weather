# rn-weather
A weather app using React Native and Context API

### How to run
1. Clone the repository
2. Put you OpenWeatherMapApi and GooglePlacesApi keys in `src/constants/keys.js`
3. Run `npm install` and install `expo-cli` if neccessary
4. Run `npm start`
